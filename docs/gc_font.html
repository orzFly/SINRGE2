Font

字体的类。字体是 Bitmap 的属性。

超类

  • Object

类方法

Font.new(name, size[, is_lattice])

生成 Font 实例。

is_lattice 为 true 时表示生成点阵字体。

Font.exist?(name)

若系统中存在指定的字体则返回 true。

属性

name

字体的名称。

该属性点阵字体无法改变。

size

字体大小，预设为 20。

该属性点阵字体无法改变。

bold

粗体标记。预设为 false。

该属性点阵字体无法改变。

italic

斜体标记。预设为 false。

该属性点阵字体无法改变。

shadow

文字阴影标记。预设为 true。设为 true 时会在文字的右下方描绘黑色的文字阴影。

该属性点阵字体无法改变。

color

文字颜色（Color）。也可以调整 alpha 值。预设为 (255, 255, 255, 255)。

alpha 值也同时用来描绘文字阴影。
